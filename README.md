# Hyderabad Globe FC - Digital Transformation 2026

## 📌 Project Overview
This is a high-performance redesign of the Hyderabad Globe FC official website. The project transitions the club from a legacy Wix site to a custom-engineered React/Tailwind application to solve issues with page speed, mobile responsiveness, and SEO discoverability.

## 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Framer Motion
- **Tooling:** Lovable.ai (Agentic Development)
- **Hosting:** Lovable Cloud / Vercel
- **SEO:** JSON-LD Schema, Semantic HTML5

## 🎯 Key Marketing Features
- **Local SEO Integration:** Content strategy focused on the Telangana Football Association (TFA) Rahim League and local summer coaching camps.
- **Performance Optimized:** 95+ Lighthouse scores for mobile and desktop.
- **AI-Readiness:** Structured data specifically designed for visibility in AI Search (Perplexity, ChatGPT, Gemini).
